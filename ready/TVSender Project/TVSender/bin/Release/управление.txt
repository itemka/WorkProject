F1 - меню выбора набора тестов
Ctrl + Backspace = F1
F2 - настройки
F3 - меню
F4 - откроет файл выделенного теста, если он является созданным
Shift + F4 - открывает директорию создаваемых тестов
F5 - открывает папку текущей операции
F6 - открывает файл текущей операции
F8 - бегунок звука показывает/скрывает
А10 - открывает список созданных тестов
F12 - забраковать
Del = F12

Enter - запуск полного тестирования (всех тестов)
Backspace - повтор теста
стрелочки вверх/вниз - выделение теста
ctrl + Enter = запуск выделенного (одного) теста
Двойной клик мыши по тесту = ctrl + Enter

в меню выбора набора тестов (F1):
Shift + Tab - отображает в виде "дерева"
